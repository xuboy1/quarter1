Ghost : Getting Started

To install this theme, you need already installed the Ghost. Here are the official Ghost guide about the Ghost installation : http://docs.ghost.org/installation/

Theme Installation

After succesfull Ghost installation, the next step is the installation of the theme.

This is what you have to do to install a new Ghost theme:

  1. Using FTP, navigate to the /content/themes/ directory and upload your theme files to this directory.
  2. Restart Ghost.
  3. Once Ghost has restarted, log into your Ghost dashboard.(your url/ghost) and go to /ghost/settings/general/.
  4. In the 'Themes' dropdown menu, select the theme you just uploaded. (Quarter)
  5. Click 'Save'.

Ghost Link on how to switching themes : http://docs.ghost.org/themes/

Theme Configuration

There are a few settings you can use to customize Quarter to your needs.

General settings

Once you are logged into your Ghost dashboard, click the Settings link. Here, you can enter your blog's title and description, as well as upload a logo and cover.

If you upload a logo, this will be displayed instead of the blog title in the header of the theme.

If you upload a blog cover, this will be displayed in the header of the theme as background.

If you upload an image as cover, be sure that is large in width, to fit all screen sizes.

User settings

You can also configure your user-specific settings, under the User tab. Here, you can upload your profile picture, an optional cover and enter your details. Your name, e-mail, bio and website link will be shown below each blog post, as well as your profile pic and cover.

Using Quarter

Quarter is designed to provide you with a rich Ghost blogging experience, and is geared towards Ghost's Markdown editor. If you're not really that familiar with Markdown, we strongly recommend taking a look at the Markdown syntax document at http://daringfireball.net/projects/markdown/syntax

Also, you can edit in Ghost admin, the post "Welcome to Ghost" to see a basic Markdown syntax.

Quarter does pack a few extra HTML classes that are designed to give you some additional options when crafting your posts.

Text alignment

There are some simple classes included for aligning your text.

Right alignment

<p class="text-right">Your text</p>

Centre alignment

<p class="text-center">Your text</p>

Floating

Left floated image

<img src="yourimage.jpg" class="alignleft"/>

Right floated image

<img src="yourimage.jpg" class="alignright"/>

Fancybox support

Fancybox support Theme includes the fancybox script if you want to link an image to fancybox. In this case, don't insert image to post via Markdown syntax but as simple HTML.

<a href="yourimage.jpg" class="fancybox"><img src="yourimage.jpg"/></a>

Here is an example : http://quarter.ghosten.com/the-following-ten-years/

Images, iframes in Index page

To displaying successfully the image or an embedded object (vimeo, youtube, soundcloud, mixcloud etc.) it is necessary to put this in the beginning of each post.

Sidebar

Ghost does not support widgets yet, but if you like to add simple HTML in sidebar the related file is in /partials theme's folder (sidebar.hbs)
